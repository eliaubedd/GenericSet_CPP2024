Filtra gli elementi di un set basandosi su un predicato fornito.

La funzione filter_out GLOBALE prende un set generico e un predicato e restituisce un nuovo set contenente gli elementi del set originale che soddisfano il predicato fornito.

Concatena due set senza duplicati.

Questa funzione prende due set di tipi generici T e li concatena, creando un nuovo set che contiene tutti gli elementi di entrambi i set di input, senza duplicati.

Template Parameters

Parameters

Returns

Note
